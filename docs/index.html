📅 Shinya Calendar

아름답고 직관적인 데스크톱 캘린더

Windows, macOS, Linux에서 사용 가능한 무료 오픈소스 캘린더 애플리케이션입니다.
Google Calendar와의 완벽한 양방향 동기화를 제공합니다.

무료 다운로드 GitHub 보기

주요 기능

강력하면서도 사용하기 쉬운 기능들

📆

다양한 캘린더 보기

월간, 주간, 일간 보기로 일정을 한눈에 확인하세요. 직관적인 UI로 쉽게 일정을 추가하고 관리할 수 있습니다.

🔄

Google Calendar 동기화

Google Calendar와 실시간 양방향 동기화. 여러 개의 캘린더를 한 곳에서 관리하고, 오프라인에서도 접근 가능합니다.

🎨

아름다운 디자인

현대적이고 깔끔한 UI/UX. 8가지 색상 팔레트로 일정을 구분하고, 다크 모드로 눈의 피로를 줄일 수 있습니다.

🔁

반복 일정

매일, 매주, 매월, 매년 반복되는 일정을 쉽게 생성하고 관리하세요. 복잡한 반복 패턴도 지원합니다.

⚡

빠른 성능

Electron 기반으로 빠르고 반응성 높은 사용 경험. 대용량 일정 데이터도 부드럽게 처리합니다.

🌍

크로스 플랫폼

Windows, macOS, Linux 모두 지원. 어떤 운영체제에서도 동일한 경험을 제공합니다.

🔐 Google Calendar 데이터 사용 정책

투명하고 안전한 데이터 처리

왜 Google Calendar 접근 권한이 필요한가요?

📥
일정 가져오기 (읽기)

사용자의 Google Calendar에 저장된 모든 캘린더와 이벤트를 읽어와 앱에서 통합하여 볼 수 있습니다. 반복 이벤트, 종일 이벤트 등 모든 유형의 일정을 지원합니다.

🔄
일정 동기화 (쓰기)

앱에서 생성, 수정, 삭제한 일정을 Google Calendar에 반영하여 모든 기기에서 일정을 최신 상태로 유지합니다.

📴
오프라인 사용

동기화된 일정을 로컬 기기에 저장하여 인터넷 연결 없이도 일정을 확인하고 관리할 수 있습니다.

데이터 보안 약속

💾
로컬 저장

모든 데이터는 사용자 기기에만 저장

🚫
외부 전송 없음

사용자 데이터를 외부 서버로 전송하지 않음

🔒
직접 통신

Google API와 직접 통신하며 중간 서버 없음

🔓
언제든 해제

설정에서 언제든지 계정 연결 해제 가능
